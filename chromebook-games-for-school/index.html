The short answer

Every WoopKid game runs in the Chrome browser on any school Chromebook, including old and slow ones. There is nothing to install, no student accounts, no sign-in and zero ads, so nothing inappropriate can appear on a classroom screen. IT only needs to allowlist a single domain: woopkid.com.

What your IT admin will ask

  • Domain to allow: woopkid.com (plus fonts.googleapis.com and fonts.gstatic.com, usually already allowed)
  • Static HTML, CSS and JavaScript only. No third-party iframes, no ad exchanges, no chat, no user accounts, no data collection
  • COPPA-friendly by architecture: there is simply nothing collected
  • The site is open source on GitHub, so district filtering teams can inspect every line

Common questions

Do WoopKid games work on managed school Chromebooks? Yes. The games are lightweight browser games that run on any Chromebook with Chrome, including managed devices, as long as woopkid.com is allowed by the school filter. No installs and no accounts are needed.

Is WoopKid safe to put on a school allowlist? Yes. The site loads no ads, no trackers and no third-party content except Google Fonts. There is no chat and no user-generated content. The full source code is public on GitHub for review.

Which games work for indoor recess? Quick games like Wing It, Sky Stack, Slice Rush and Tap Rush fit short breaks. Bubble Tank Duel lets two students share one keyboard, which makes it a favorite for indoor recess.
